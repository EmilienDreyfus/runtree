# Code of Conduct

This project expects contributors to act professionally and respectfully.

## Expected behavior

- Be direct, constructive, and respectful.
- Assume good intent, but focus on evidence and reproducibility.
- Keep technical disagreement about the work, not the person.
- Respect maintainer time and project boundaries.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks
- Doxxing or threats
- Spam or repeated bad-faith engagement
- Deliberate disruption of issues, pull requests, or reviews

## Enforcement

The maintainer may edit, remove, lock, or close content, and may restrict participation for behavior that violates this policy.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project communication channels.
